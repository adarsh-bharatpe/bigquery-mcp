# BigQuery / Metabase context

This folder holds notes and reference material for **Metabase tables** (and related BigQuery sources) grouped by app.

## Layout

| Folder | App |
|--------|-----|
| `bharatpe-consumer/` | BharatPe consumer |
| `invest-bharatpe/` | Invest BharatPe |
| `bharatpe-merchant/` | BharatPe merchant |

Add per-app markdown files here (for example: table lists, field meanings, join patterns, known caveats). Keeping one app per directory avoids mixing contexts when working with BigQuery MCP or Metabase.

## Global vs per-app docs

- **This README** — high-level index and conventions only.
- **Subfolders** — detailed table/schema context for that product.
